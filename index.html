Minggu 1 — Core Laravel & Database Tersedia

Instalasi Laravel (XAMPP & WSL), Eloquent & relasi model, migration, optimasi query dasar (N+1 & indexing), keamanan dasar (CSRF, XSS, SQL Injection), REST API, dan autentikasi JWT.

  • Modul utama — Instalasi (XAMPP/WSL), CRUD, Eloquent & Relasi, Query Optimization, REST API + Sanctum, Keamanan
  • Lanjutan — Workshop autentikasi API dengan JWT

Minggu 2 — Arsitektur Kode & Otorisasi Tersedia

Repository/Service Pattern, Middleware & Validation, Roles & Permission (Spatie), Rate Limiting.

  • Modul — Repository/Service Pattern, Custom Middleware & Form Request, Spatie Roles & Permission, Rate Limiting

Minggu 3 — Testing & Optimasi Tersedia

Pest/PHPUnit, Redis (cache & session), Laravel Horizon (queue).

  • Modul — Redis, Laravel Horizon, Pest/PHPUnit

Minggu 4 — Observability Tersedia

Laravel Telescope, Logging (Monolog), Error Tracking (Sentry).

  • Modul — Laravel Telescope, Logging & Monolog Channel, Error Tracking dengan Sentry

Minggu 5 — API Design & Real-time Tersedia

OpenAPI/Swagger, API Versioning, WebSocket/Broadcasting (Laravel Reverb).

  • Modul — API Versioning, Dokumentasi OpenAPI/Swagger, Real-time dengan Laravel Reverb

Minggu 6 — Security Deep-dive Tersedia

OWASP Top 10, Encryption & Secrets Management, API Security Hardening.

  • Modul — OWASP Top 10, Encryption & Secrets Management, API Security Hardening

Minggu 7 — Capstone Project: E-commerce Tersedia

Bangun API e-commerce lengkap (produk, cart, checkout, payment gateway, order management) menerapkan skill Minggu 1–6.

  • Modul — Shopping Cart, Checkout & Order State Machine, Payment Gateway (Midtrans), menggabungkan skill Minggu 1–6

Minggu 8 — Containerization & Deployment Tersedia

Docker, CI/CD (GitHub Actions), k6 (Load Testing) — diterapkan ke project e-commerce.

  • Modul — Install Docker Desktop, Dockerize Laravel, CI/CD dengan GitHub Actions, Install & jalankan k6 Load Testing

Minggu 9 — Cloud & Infrastructure Tersedia

Nginx/Supervisor, Cloud (AWS/GCP), Infrastructure as Code (Terraform) — deploy project e-commerce.

  • Modul — Install & konfigurasi Nginx/Supervisor, Install CLI AWS/GCP, Install Terraform, Deploy E-commerce ke Cloud

Minggu 10 — Skala Besar Tersedia

Microservices Architecture, Message Brokers (Kafka/RabbitMQ), CQRS & Event Sourcing, Kubernetes.

  • Modul — Microservices, Install RabbitMQ/Kafka, CQRS & Event Sourcing, Install kubectl/Minikube

Minggu 11 — System Design Tersedia

Database Sharding & Replication, Load Balancing, Distributed Systems (CAP Theorem).

  • Modul — Database Replication & Sharding, Load Balancing (Nginx), Distributed Systems & CAP Theorem
🎉 Roadmap 11 Minggu Selesai!
Semua modul dari Minggu 1 sampai Minggu 11 sudah tersedia — dari instalasi Laravel pertama sampai memahami CAP Theorem. Mulai dari Minggu 1 kalau baru mulai, atau langsung ke modul mana pun lewat menu Laravel di atas.
Cara Membaca Online via GitHub
Push folder ini ke repository GitHub, lalu aktifkan GitHub Pages (Settings → Pages → Deploy from branch) supaya semua modul .html bisa dibuka langsung dari browser tanpa server lokal. Lihat README.md untuk detailnya.

Git — Version Control Tersedia

Panduan lengkap Git dari pemula sampai profesional: konsep dasar & alur perpindahan file (dengan visualisasi Mermaid.js), branching & merging, kolaborasi lewat remote, strategi branching (Git Flow/GitHub Flow/Trunk-based), teknik lanjutan (rebase, cherry-pick, stash), sampai best practice tim profesional.

  • Modul — Instalasi & Konfigurasi, Alur Kerja, Branching & Merging, Remote & Kolaborasi, Strategi Branching, Teknik Lanjutan, Best Practice
